# EduRetoSV – Plataforma de Juegos Educativos

Proyecto web estático para niños y niñas que desean aprender mediante juegos interactivos.

## Estructura del Proyecto

```
EduRetoSV/
├── index.html          # Página principal
├── materias.html       # Página de materias
├── matematica.html     # Página de matemáticas
├── matjuego1.html      # Juego 1 de matemáticas
├── ciencia.html        # Página de ciencias
├── gramatica.html      # Página de gramática
├── ingles.html         # Página de inglés
├── login.html          # Página de login/registro
├── css/                # Archivos CSS
├── js/                 # Archivos JavaScript
├── img/                # Imágenes
└── assets/             # Assets de juegos (audio, imágenes de juegos)
```

## Cómo usar

Simplemente abre los archivos HTML en tu navegador:

1. Abre `index.html` en tu navegador
2. Navega por las diferentes páginas usando los enlaces
3. Juega los juegos educativos disponibles

## Archivos principales

- **index.html**: Página de inicio con información sobre la plataforma
- **materias.html**: Vista de todas las materias disponibles
- **matematica.html**: Página de matemáticas con acceso a juegos
- **matjuego1.html**: Juego de secuencias numéricas
- **login.html**: Página de registro e inicio de sesión (sin funcionalidad de backend)

## Notas

- Este es un proyecto estático, no requiere servidor
- Los juegos funcionan completamente en el navegador
- No hay sistema de base de datos ni autenticación funcional
- Todos los archivos estáticos están organizados en carpetas: `css/`, `js/`, `img/`, `assets/`
